# ProjectVault 팀 회의록

## 회의 목적

과제 요구사항을 읽고 ProjectVault Agent의 MVP 범위를 정했다.

## 결정사항

- Obsidian Markdown 출력은 필수 기능으로 구현한다.
- Google Sheets 저장은 분석 결과를 표 형태로 확인하기 위해 필수 기능으로 구현한다.
- NotebookLM처럼 질문에 답하는 도구가 아니라, 입력 자료를 분석해서 작업공간을 생성하는 에이전트로 설명한다.
- 평가 시연에서는 실제 API 키 없이 dry-run이 돌아가야 한다.
- write 모드에서는 output_vault에 Markdown 노트, graph.json, sheets_rows.json이 생성되어야 한다.
