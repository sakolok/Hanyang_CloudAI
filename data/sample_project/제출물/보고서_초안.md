# ProjectVault Agent 보고서 초안

## 1. 프로젝트 개요

ProjectVault Agent는 과제 공지, 강의자료, 코드, 메모, 오류 로그를 읽고 Obsidian 작업공간과 graph.json, Google Sheets 요약표를 생성하는 에이전트다.

## 2. 에이전트 필요성

학생 프로젝트는 과제 요구사항, 강의 개념, 코드 오류, 회의 결정사항이 여러 파일에 흩어져 있다. ProjectVault는 이를 하나의 작업공간으로 정리해 다음 행동을 빠르게 확인하도록 돕는다.

## 3. 워크플로우

입력 폴더를 스캔한 뒤 파일별 분석을 수행하고, 그래프 노드와 엣지를 만든다. 이후 Obsidian Markdown 노트와 Google Sheets rows를 생성한다.

## 4. 구현 방법

LangGraph StateGraph를 사용해 scan, analyze, build graph, export notes, prepare sheets, report 단계를 연결한다.
