# 오류 로그와 해결 계획

## PDF 텍스트 추출 실패 가능성

일부 강의자료 PDF가 이미지 기반이면 pypdf로 텍스트가 추출되지 않을 수 있다. MVP에서는 OCR을 제외하고, 텍스트 추출이 비어 있으면 warning을 남긴다.

해결 방안:
- PDF 텍스트가 비어 있는 경우 Error 노트에 기록한다.
- OCR은 향후 개선 항목으로 분리한다.
- 텍스트 버전 강의 노트를 함께 넣으면 시연이 안정적이다.

## Google Sheets 인증 실패 가능성

서비스 계정 파일 경로가 없거나, Google Sheet가 서비스 계정 이메일에 공유되지 않으면 append가 실패할 수 있다.

해결 방안:
- 인증 실패 시 workflow를 중단하지 않는다.
- local fallback으로 sheets_rows.json을 생성한다.
- README에 Google Cloud Console 설정 순서를 작성한다.

## 보안 위험

API 키나 토큰이 코드 또는 제출물에 들어가면 감점 위험이 있다. Secret scanner가 위험 파일명과 secret-like pattern을 탐지해서 Error와 Task로 기록해야 한다.
