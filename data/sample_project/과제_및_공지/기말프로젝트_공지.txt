ProjectVault Agent 과제 공지

이번 과제의 목표는 LangGraph와 LangChain을 활용해 여러 입력 자료를 분석하고, 프로젝트 작업공간을 자동 생성하는 에이전트를 설계하고 구현하는 것이다. 단순 요약 챗봇이 아니라 입력 폴더를 읽고 Obsidian Markdown 노트, 작업 그래프, Google Sheets 요약표를 생성하는 흐름을 보여야 한다.

보고서 요구사항
- 보고서는 최소 A4 5페이지 이상으로 작성한다.
- 보고서 첫 부분에는 팀원 이름과 소속 대학을 기재한다. 개인정보 보호를 위해 공개 시연 자료에는 예시 이름을 사용하고 실제 학번은 쓰지 않는다.
- 에이전트가 필요한 이유, 목적, 전체 개요를 설명한다.
- 입력 폴더 스캔부터 파일 분석, 그래프 생성, Obsidian export, Google Sheets 저장까지의 워크플로우를 단계별로 설명한다.
- 실행 예시 입력과 출력 예시를 반드시 포함한다.
- 구현 방법에는 LangGraph State, Node, Edge 구성과 LangChain/Gemini API 사용 방식을 포함한다.
- 소스코드 전체를 제출한다.
- GCP 또는 Google API를 하나 이상 활용해야 한다. 이 프로젝트에서는 Google Sheets API 저장 기능을 사용한다.
- API 키, 서비스 계정 파일, 토큰 파일은 제출물에 포함하면 안 된다. 환경변수와 .gitignore 설정을 사용한다.

평가 기준
- 아이디어와 문제 정의: 70%
- 구현 완성도와 시연 가능성: 30%

제출물
- 보고서 PDF
- 소스코드 압축 파일 또는 저장소 링크
- 실행 방법 README
- 샘플 입력 폴더와 생성된 출력 예시

주의사항
- 실제 API 키나 개인정보를 보고서, 코드, 샘플 데이터에 넣지 않는다.
- 서비스 계정 JSON 파일은 로컬에만 보관하고 Google Sheet에는 서비스 계정 이메일을 공유한다.
- 오류 로그와 해결 과정을 보고서에 간단히 포함한다.
